## Selection Criteria

When adding components to this catalog:
1. Must be open source or have a compatible license
2. Must meet WCAG 2.1 AA accessibility standards (or be fixable)
3. Must be adaptable to the AURORA design token system
4. Must work within Streamlit constraints (or be adaptable)
